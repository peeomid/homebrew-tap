# Homebrew Tap

## Available Formulas

### todosmd

A terminal-based task manager for Markdown files with vim keybindings.

```bash
brew tap peeomid/tap
brew install todosmd
```

After installation, use the `tmd` command:

```bash
tmd --help
tmd init
tmd interactive
```
